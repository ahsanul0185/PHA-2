# Vehicle Rental System

A Vehicle Rental System built with **Express**, **TypeScript**, and **PostgreSQL** for managing vehicles, customers, and bookings efficiently.

---

## Live Deployment
https://vehicle-rental-system-ecru.vercel.app/

---

## Features

- User authentication and authorization
- Manage customers and vehicles
- Create, view, and manage bookings
- RESTful API design
- Input validation and error handling
- PostgreSQL database integration

---

## Technology Stack

- **Backend:** Node.js, Express, TypeScript  
- **Database:** PostgreSQL  

---

## Setup & Usage

### 1. Clone the repository
```bash
git https://github.com/ahsanul0185/PHA-2.git
cd PHA-2
````

### 2. Install dependencies

```bash
npm install
```

### 3. Configure environment variables

Create a `.env` file in the root directory:

```
PORT=5000
DB_CONNECTION_STRING=postgresql://username:password@localhost:5432/yourdb
JWT_SECRET=your_jwt_secret
```

### 4. Run the project

For development:

```bash
npm run dev
```

For production:

```bash
npm run build

```
